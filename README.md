# Маяк — Трекер привычек

Мобильное приложение (web/PWA) бренда «Коучинг-Маяк»: 6 направлений жизни, фигура состояния (звезда/радар), ежедневные галочки, квартальные нормы.

## Архитектура
Микрофронтенд **app-shell**: каждая вкладка — изолированный iframe-модуль. Сбой одной вкладки не роняет shell и другие.

```
index.html              app-shell: каркас iPhone 15 Pro, таб-бар (4 вкладки), iframe-stage, error-boundary
tabs/radar.html         Вкладка 1 (главная): фигура + баллы дня + тап-сектор → галочки → «закрытие»
tabs/soon2.html         Вкладка 2: Итоги (Скоро)
tabs/soon3.html         Вкладка 3: Задачи (Скоро)
tabs/profile.html       Вкладка 4: Личный кабинет (Supabase Auth — Фаза 2)
lib/supabase.js         клиент Supabase (Фаза 2)
manifest.webmanifest    PWA
```

## Изоляция
Каждый модуль шлёт `postMessage({type:'tab-ready', tab})` родителю при загрузке. Shell ждёт сигнал ~4с; если нет — показывает фолбэк-плашку, **не блокируя таб-бар и другие вкладки**.

## Стек
- Фронт: статические HTML/JS, без сборки. Деплой — Vercel.
- Бэкенд (Фаза 2): Supabase (Postgres + Auth + авто-бэкапы + RLS).

## Статус
- Фаза 1: app-shell + 4 вкладки (Радар рабочий, 2/3 «Скоро», Кабинет-заглушка). Данные локально.
- Фаза 2: Supabase (данные/авторизация/бэкапы).
